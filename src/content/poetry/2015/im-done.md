title=I'm Done
date=2015-07-19
type=post
cat=poetry
tags=2015,poetry,love

~~~~~~
I'm done.
Done with trying to find love.
Done with worrying about others.
Done with caring.
Done.
I'll never understand this culture.
The careless attitudes.
The games.
The hate.
So, I'm done.
Let me live my life, please.
Or not, but make it quick.
Because, I'm done.
